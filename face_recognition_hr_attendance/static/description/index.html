HR Attendance Face Recognition

Employee Attendance using Face Recognition

Few Words About The HR Attendance Face Recognition:

This module will help you to convert your normal attendance system to facial recognition attendance system. To use module you have to configure the employees images only, those images will be used for face recognition. This app is capable to work with large databases.

Main features:

  • Face Recognition works on attendance kiosk mode, and it is responsive for different screen sizes.
  • Also works on Android devices
  • We can add multiple images for an employee for better recognition
  • Direct capture employee images from webcam
  • It used prebuilt recognition model, this will increase the recognition speed
  • Do not allow unknown faces
  • It uses server side face recognition for better results
  • In the attendance module settings you can also control the attendance mode, Enable the mode that you want to use
  • In the attendance record it will save the check-in and check-out attendance image
